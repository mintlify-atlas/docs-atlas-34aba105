## Secure Your Server Commands

SecureCommands is a powerful Spigot/Paper plugin that adds an authentication layer to critical server commands, protecting your Minecraft server from unauthorized access with BCrypt-encrypted passwords and whitelist-based access control.

## How It Works

SecureCommands protects your server by requiring players to authenticate before executing blocked commands. Administrators can manage a whitelist of trusted players, configure which commands to protect, and receive instant Discord notifications when security events occur.

<Steps>
  <Step title="Install & Configure">
    Install the plugin, configure your MySQL database, and define which commands to protect
  </Step>
  <Step title="Add Players to Whitelist">
    Add trusted players to the whitelist and set up their authentication codes
  </Step>
  <Step title="Players Authenticate">
    Players use `/sclogin` to authenticate before executing protected commands
  </Step>
  <Step title="Monitor Security Events">
    Receive Discord notifications for failed login attempts and blocked commands
  </Step>
</Steps>
